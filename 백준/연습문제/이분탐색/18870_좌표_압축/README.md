## 문제

수직선 위에 N개의 좌표 X_(1), X_(2), ..., X_(N)이 있다. 이 좌표에 좌표 압축을 적용하려고 한다.

X_(i)를 좌표 압축한 결과 X&#39;_(i)의 값은 X_(i) > X_(j)를 만족하는 서로 다른 좌표 X_(j)의 개수와 같아야 한다.

X_(1), X_(2), ..., X_(N)에 좌표 압축을 적용한 결과 X&#39;_(1), X&#39;_(2), ..., X&#39;_(N)를 출력해보자.

## 입력

첫째 줄에 N이 주어진다.

둘째 줄에는 공백 한 칸으로 구분된 X_(1), X_(2), ..., X_(N)이 주어진다.

## 출력

첫째 줄에 X&#39;_(1), X&#39;_(2), ..., X&#39;_(N)을 공백 한 칸으로 구분해서 출력한다.
