## 문제

무한 수열 A는 다음과 같다.

	A_(0) = 1
	A_(i) = A_(&lfloor;i/P&rfloor;) + A_(&lfloor;i/Q&rfloor;) (i ≥ 1)

N, P와 Q가 주어질 때, A_(N)을 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 3개의 정수 N, P, Q가 주어진다.

## 출력

첫째 줄에 A_(N)을 출력한다.
