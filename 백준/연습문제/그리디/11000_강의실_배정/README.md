참고로, 수업이 끝난 직후에 다음 수업을 시작할 수 있다. (즉, T_(i) ≤ S_(j) 일 경우 i 수업과 j 수업은 같이 들을 수 있다.)

수강신청 대충한 게 찔리면, 선생님을 도와드리자!
